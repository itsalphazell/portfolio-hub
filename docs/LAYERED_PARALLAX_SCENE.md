# Layered Parallax Scene

Source inspiration: `osmosupply/NWQevrB`

Use this module when the page needs more spatial depth and premium scroll feel, but a full immersive canvas would be too heavy.

Use it for:

- marketing home pages
- story sections
- campaign landings
- showcase pages with layered media

Do not use it for:

- dense pricing or docs pages
- product application surfaces
- pages already running a full 3D hero

Files:

- `react/examples/LayeredParallaxScene.tsx`
- `static/html/layered-parallax-scene.html`
- `static/css/layered-parallax-scene.css`
- `static/js/layered-parallax-scene.js`

Expected effect:

- layered depth
- premium scroll feel
- atmospheric motion without a full scene runtime
