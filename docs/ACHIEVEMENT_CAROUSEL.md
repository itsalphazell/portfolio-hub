# Achievement Carousel

Source inspiration: `Karthigaiselvam-dev`

Use this module when proof, certifications, milestones, or awards need stronger pacing than a static badge grid but should remain easier to maintain than a custom scene.

Use it for:

- certifications and training sections
- trust or proof blocks
- portfolio milestones
- premium resume and founder sites

Do not use it for:

- long lists of badges
- dense compliance matrices
- pages where the content must remain simultaneously visible

Files:

- `react/components/AchievementCarousel.tsx`
- `static/html/achievement-carousel.html`
- `static/css/achievement-carousel.css`
- `static/js/achievement-carousel.js`

Expected effect:

- spotlight-style proof presentation
- stronger visual pacing for technical trust content
- more premium proof blocks without a heavy dependency
