# Pointer Media Reveal

Source inspiration: `Hyperplexed/wvQzbGz`

Use this pattern when one visual should react to pointer intent and give a premium feel without pushing the page into a full immersive build.

Use it for:

- campaign intros
- feature reveals
- showcase sections

Do not use it for:

- utility surfaces
- mobile-first sections with no clear hover fallback
- pages already carrying multiple spectacle moments

Files:

- `react/components/PointerMediaReveal.tsx`
- `static/html/pointer-media-reveal.html`
- `static/css/pointer-media-reveal.css`
- `static/js/pointer-media-reveal.js`

Expected effect:

- pointer-led image reveal
- stronger interaction without scene overhead
- effect-bank quality that still stays usable in production
