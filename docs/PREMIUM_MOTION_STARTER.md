# Premium Motion Starter

This kit installs reusable premium motion building blocks for React and static projects.

## Included
- premium text reveal patterns
- GSAP and SplitType helpers
- static and React examples
- reduced-motion-safe timing defaults
- editorial showcase hero presets for React and static sites
- product blur slider preset for React and static sites
- cyber terminal hero preset for React and static sites
- perspective tilt headline snippet
- pointer media reveal snippet
- card showcase stack snippet
- achievement carousel module for React and static sites

## Use
- import the React assets from `codex-premium-motion/react/`
- or adapt the static assets from `codex-premium-motion/static/`
- keep motion focused on hero, section intros, and key proof reveals
- for AWWWARDS-like premium web composition, start from the editorial showcase hero preset before inventing a new hero structure
- for campaign or product pages, start from the product blur slider preset before adding a generic carousel library
- for technical portfolios or security-led personal sites, start from the cyber terminal hero preset before layering ad hoc terminal effects into a generic hero
- for lighter interaction-driven sections, use pointer media reveal or card showcase stack before escalating into immersive modules
- for trust, certifications, or proof sections, use achievement carousel before falling back to a dead logo wall
