# Card Showcase Stack

Source inspiration: `filipz/bNNjwdE`

Use this pattern when a section needs depth and hierarchy but a full immersive stage would be excessive.

Use it for:

- campaign sections
- product proof sequences
- editorial showcase blocks

Do not use it for:

- data-dense content
- docs or pricing tables
- pages already dominated by a 3D hero

Files:

- `react/components/CardShowcaseStack.tsx`
- `static/html/card-showcase-stack.html`
- `static/css/card-showcase-stack.css`
- `static/js/card-showcase-stack.js`

Expected effect:

- premium depth from a simple card stack
- stronger section pacing
- more visual structure without adding a new runtime dependency
