# Premium Static Parallax Story

Use this preset when the project is static or template-driven and you want one immersive scroll section without pulling the whole page into a heavier React/3D stack.

## Source profile

This preset is extracted from the reusable qualities of `Parallax-website`:

- one strong hero statement
- layered depth coefficients
- readable text reveals
- a static-first architecture that still feels premium

## Best fit

- static landing pages
- product or service showcases
- premium campaign pages

## Rules

1. Keep the copy readable before scroll motion is added.
2. Use only a few depth layers with clear coefficients.
3. Do not turn the whole page into an effect reel.
4. Prefer graceful fallback when GSAP is not present.
5. Keep reduced-motion behavior intact.
