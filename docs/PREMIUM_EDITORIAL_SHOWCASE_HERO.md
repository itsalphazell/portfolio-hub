# Premium Editorial Showcase Hero

Use this preset when the target is a premium portfolio, agency site, or founder landing page that should feel editorial and polished before it feels immersive.

## Source profile

This preset is extracted from the reusable patterns seen in `AWWWARDS-Standard_Portfolio-Website`:

- large editorial typography
- strong section rhythm
- one optional visual accent instead of a scene-heavy page
- premium motion driven by GSAP and text choreography
- optional smooth-scroll pairing without making scroll tech the whole point

## Best fit

- premium-web marketing pages
- service portfolios
- agency and founder sites
- premium-immersive-lite pages that want one light visual accent

## Use it with

- `premium-motion-system`
- `smooth-scroll-pack` if the page is marketing-led
- `shader-pack` only if the still frame improves

## Do not use it for

- app dashboards
- dense product surfaces
- pages that already rely on a full 3D scene

## Key rule

The hero must still work as a still frame with excellent copy hierarchy, even if you later add a light 3D accent or smooth-scroll polish.
